# TwitterAPI.io Guidance

Use `twitterapi_advanced_search` for X/Twitter public tweet search. Include `since:YYYY-MM-DD` in the query string for last-30-days style research.

Pricing note: TwitterAPI.io public pricing is $0.15 per 1,000 returned tweets, with a $0.00015 minimum per request. Bound query branches and time windows before fanout.
